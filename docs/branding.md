# Branding — Voisin D'abord

## Positionnement
Service à domicile local, porté par une personne réelle (pas une agence, pas une plateforme). Le différenciateur central : la relation directe, l'honnêteté, un seul interlocuteur. Le ton est celui d'un voisin de confiance, pas d'une entreprise.

## Palette de couleurs (tokens actifs dans les `<style>` inline)

| Token          | Valeur    | Usage                                    |

| `--vert`       | `#2d5a27` | Couleur principale, CTA, accents         |

> `style.css` contient une autre palette (`--texte`, `--or`, Cormorant Garamond) qui n'est plus utilisée — ne pas s'y référer.

## Typographie
- **Titres / accents** : Fraunces (serif) — poids 300, 400, 600 ; italique pour les mots clés colorés en vert
- **Corps / UI** : DM Sans — poids 300, 400, 500
- **Taille de base** : 16px, line-height 1.6–1.85 selon contexte

## Ton éditorial
- Première personne du singulier ("je viens", "je m'occupe", "je vous appelle")
- Direct, sans jargon, sans sur-promesse
- Authenticité > professionnalisme froid
- Répéter régulièrement les deux arguments clés : **tarif transparent** + **crédit d'impôt 50 %**

## Éléments à ne pas changer
- Le nom "Voisin D'abord" : fort, local, mémorable
- L'apostrophe dans le nom est intentionnelle (D'abord = avant tout) — le domaine `voisindabord.fr` est sans apostrophe
- La structure vert foncé / crème chaud : distinctive et cohérente avec le positionnement "nature, local, bienveillant"
